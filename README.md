# google-finance
Currency converter API written in NodeJS using Express.js

## Live
This package is hosted at http://pub.ioncodes.com:3000/

Simply do this:
http://pub.ioncodes.com:3000?from=BTC&to=USD&amount=2

## npm
Install via npm:
´´´
npm install google-finance-api
´´´
